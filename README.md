# Videoplayer

Учебный проект с курса [Devman.org](https://devman.org).

Кастомный видеоплеер на [Playable.js](https://playable.io/).

![скриншот плеера](screenshot.png)

## Требования

```bash
pip install livereload
```

## Как запустить

```bash
python server.py
```

Откройте `http://127.0.0.1:5500/` в браузере.
